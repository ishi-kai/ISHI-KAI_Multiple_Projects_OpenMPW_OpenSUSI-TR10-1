## 絶対値回路
絶対値回路は、ソースフォロワでは、入力電圧オフセットの問題をどうやっても解決できなくて、電流制限型ゲート接地回路＋ゲート電圧オフセット回路で解決しました。ゲート電圧オフセット用に自分自身と同じサイズのトランジスタを別に作って、絶対値回路としての入力電圧オフセットをキャンセルするためのバイアス電圧を生成しています。入力インピーダンスは低いと思われる＆入力電圧の2.5Vから上の数１０mV程度は不感帯がどうしても残ります。ゲート接地回路にしたことにより、この回路自体には増幅機能が全くありませんので、前段のRFアンプは頑張って増幅率を稼ぐ必要ある仕様となっております。
