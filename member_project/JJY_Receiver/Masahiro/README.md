# 概要

耳で聞けるJJY（標準電波）受信機のICを自作する設計データリポジトリ。目標はスピーカから毎時15分と45分に流れる「JJY JJY」のコールサインを耳で聞くこと。あわよくばタイムコードを復号して7セグメント表示機で表示できるようにする。

# 設計制約
## 使用するPDK
OpenSUSI for TokaiRika Shuttle PDK (OpenSUSI-TR10)

## 面積・ピン数
- 面積: 2000um x 1000um
- ピン数: 10ピン (VDD込み・VSSは共通)
