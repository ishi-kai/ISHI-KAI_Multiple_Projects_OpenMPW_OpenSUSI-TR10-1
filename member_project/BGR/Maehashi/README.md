# Bandgap Reference Circuit
[Open Source Silicon Magazine Vol.2](https://github.com/ishi-kai/OpenSourceSiliconMagazine/tree/main/Vol.2)に搭載している参照電流源・電圧源回路を、PVTばらつきを考慮してチューニングしたものを実際に設計したもの。

### bgr_a
PTAT電流を生成し、ダイオードに流して定電圧を得る回路

### bgr_b
定電流を生成し、抵抗に流して定電圧を得る回路（定電流はそのままカレントミラーして電流出力にもしている）
